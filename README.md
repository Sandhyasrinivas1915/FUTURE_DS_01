# Business Sales Performance Analytics

This project analyzes business sales data to identify revenue trends, top-selling products, high-value categories, and regional performance.

Tools Used:
- Microsoft Excel
- Pivot Tables
- Data Visualization

Key Insights:
- West region generates the highest revenue.
- Technology category contributes the most sales.
- Canon imageCLASS 2200 Advanced Copier is the top-selling product.
- Sales peak during November and December.

Recommendations:
- Increase marketing during high-performing months.
- Promote technology products more.
- Maintain higher stock for top-selling products.
